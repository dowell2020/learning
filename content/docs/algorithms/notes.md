# 每周一题

每周一道算法题。

## 规则

Go 夜读知识星球每周四晚上会发布一道算法题，周日晚上22点截止提交。

### 参与方式：

1. 知识星球打卡参赛；
2. GitHub Pull Request 参加；

### 奖励（Go 夜读知识星球会员）：

1. 参与奖：1 元；
2. 优秀奖：20 元；

**注：**：组委会将会根据实际情况，保留对奖励分配的最终解释权。
